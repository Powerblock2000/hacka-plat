# Hacka-Plat
![Hackaplat Banner Logo](https://media.githubusercontent.com/media/Powerblock2000/hacka-plat/refs/heads/main/Textures/Banner.png)

Welcome to Hacka-Plat, a game about solving puzzles by "hacking" (moving) platforms!

## Controls
> Use ↑ (Up arrow) and ↓ (Down arrow) to navigate menus
> 
> Use [A] and [D] to move left and right
> 
> Use [Space] or [W] to jump
> 
> Press [E] to enter edit mode
> 
> Press [Shift] to dash (Left and Right ONLY)
> 
> Check the bottom of the screen for your player's current status

## Features
> 
> Basic platform movement
> 
> Dashing with re-gen timer
> 
> Moving specific platformers
